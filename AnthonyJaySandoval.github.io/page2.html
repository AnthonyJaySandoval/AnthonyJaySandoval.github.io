The project I have chosen is an Earthquake map. This project allows the user to view earthquakes that have occured recently around the world. This project uses Google Map APIs to display the magnitude and location of each earthquake. There is a map of the world that the user can look through, and when the user clicks on one of the markers, it gives a brief description of the earthquake. This project uses the U.S. Geological Survey to map the earthquakes, and their website can be found here.
